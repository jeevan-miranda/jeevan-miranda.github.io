Javascript Fundamentals

What is Javascript?

Javascript is a coding language that makes web pages interactable. It can give buttons and writeable text boxes functionality by causing changes in the webpage when users interact with those webpage elements.
It also allows for other dynamic features like videos, slideshows and animated graphics to be implemented in a webpage.

How does it relate to HTML and CSS

HTML, CSS and Javascript work togeether to make the foundation of web development.

  • HTML contains all of the headers, text any any images you want to display in the webpage.
  • CSS controls how the page looks, such as modifying fonts, background colors and where each HTML element sits on the webpage.
  • Javascript makes the webpage come alive by letting users engage with the different elements of your webpage.

An analogy that best describes the relationship would be that of kitchen:
HTML is the language that would determine what applicances and furniture would be in the kitchen
CSS would then work to arrange the design of the kitchen, detemining where applicances like the fridge and toaster should be placed, and also deciding what colour the kitchen tiles should be. Overall it works to make the kitchen look pretty.
Javascript would then be the language that makes all the appliances in the kitchen work, e.g making sure that the toaster is plugged in so that when a user presses down on the toaster handle, it starts the heating element.

Control Flows

Control flow allows our code to make decisions on what code is executed and when. It allows developers to specify what parts of thecode are run and under what conditions.
The easiest way to understand how a control flow works is by looking at a flowchart. When following a flowchart, each decision can lead to a different outcome as it determines what decision or outcome comes next. Similarly, control flow allows developers to set conditions that need to be met for the code to execute to achieve the appopriate outcome.

Conditional statements:

Conditional statements are basically checks to see if a certain condition is either true or false. If a condition is true the run code A, it it's false run code B. For example in the flowchart Fig.1 the question Is it alive? has two possible answers and then if the condition is true (yes) it runs code A (which triggers the next condition 'Should it be moving'), or false (no) it runs code B (providing outcome "Don't hit it").

Loops

Loops are a form of conditional statement that repeats a task until the condition required has been met.
The flowchart to the right displays an illustration of a loop. In this illustration, we're trying to boil water. After initiating the loop, it will start the task "wait 2 minutes" then check if the water has reached 85°C. If the condition hasn't been met yet, it then repeats the task to wait another two minutes and checks if the water has met the condition (reached 85°C). It will keep repeating the task until the condition is met. If it has reached that temperature, the loop ends.

The For loop is an extremely common type of loop structure. It allows for a task to be repeated until the conditions set in the loop have been met.

Syntax:

for (initialExpression; condition; updateExpression) {
// for loop body
}

for refers to the conditional loop statement
initialExpression is the inital variable from which the execution of the loop begins
condition is where the conditions that need to be met are written
updateExpression is where the initial variable is increased or decreased.

When executed, the control moves to the updateExpression and gets updated and then moves to the condition. If the updated variable doesn't meet the condition the loop will continue until the updated variable meets the condition.

The Document Object Model (Dom)

The DOM is a javascript object that represents the page you see in the web browser. When you make changes to the javascript model in the DOM, it will automatically update your web browser page.
When a webpage is loaded, the browser constructs the webpage HTML as a javascript object in the DOM, and structures the elements in a tree-like representation. The DOM can be then be used to find specific elements within the document, and then manipulate it using Javascript to change it.

Arrays

An array is an ordered collection or list of values stored within a single variable. each value in an array is numbered, where the first value in the array is numbered 0, the second value is numbered 1, the thrid value is numbered 2 and so on.

To declare an array, you can use the syntax:

let arrayName = [value1, value2, value 3, etc...]

You can access a specific value from the array using the number assigned to the value in the array

For example:

//create an array
let sweets = ["lollipop", "liquorice", "sherbet"]

//access the array element using console.log
console.log(sweets[0])

As mentioned before, as the first value in our array is numbered 0, searching for this value number when console logging will return the value name from the array.

Objects

An object is a collection of related properties or methods (functions that belong to an object). Within the object, all the properties and methods are stored, and to access an object property in the DOM, a user can access the object property name to access the information entered in that property

To declare an object, you can use the syntax:

let car = {
make: "toyota"
model: "camry"
color: "blue"
}

In the example above, the make, model and color are all properties, and 'toyota', 'camry', 'blue' are all values assigned to each property.

To then access a property value, you can use the property name in the console log

//access the object element using console.log
console.log(car.make)

Functions

Functions arem a section of re-useable code. It's useful for performing specific tasks at different stages of a webpage.
Once you declare a function, it can be used whenever you want by calling the function.

An easy way to think of it would be like setting up a shortcut key on your computer for a video game. Once you set up the shortcut, each time you press the shortcut key (calling the function) it accesses the menu or item in the videogame, saving you have to navigate manually.

To declare a function, you can use the 'function' keyword followed by the function name and then () brackets. For example:

//function to add two numbers when called//
function add(x, y) {
let result = x + y
return result
}

Note in this example, person and age are parameters set within the parenthesis.

To call a function, you then need to enter the function name, followed by the arguments you want to assign to the parameters. e.g
add(2, 3)
